PottsToolkit API Reference

Full docstring index for the PottsToolkit package. Note that PottsToolkit re-exports CorePotts's public API; the symbols listed here are those defined in PottsToolkit itself. For CorePotts types and functions see the CorePotts API Reference.

PottsToolkit.System.AdhesionComponent — Type
AdhesionComponent(mappings::Pair{Tuple{CellType, CellType}, Float32}...)

Maps a pair of CellTypes to their adhesion penalty (J-value).

PottsToolkit.System.CellType — Type
CellType(name::Symbol)

Abstract representation of a cell's identity.

PottsToolkit.System.ChemotaxisComponent — Type
ChemotaxisComponent(mappings::Pair{CellType, <:Real}...; chemical_field)

Maps a CellType to its chemotactic sensitivity λ across a given chemical_field.

PottsToolkit.System.HSTVolumeComponent — Type
HSTVolumeComponent(mappings::Pair{CellType, <:NamedTuple}...; eta=1.0)

Maps a CellType to its hydrostatic (target, λ) volume parameters.

PottsToolkit.System.LengthComponent — Type
LengthComponent(mappings::Pair{CellType, <:NamedTuple}...; eta=1.0)

Maps a CellType to its hydrostatic (target, λ) major-axis length parameters for cell elongation.

PottsToolkit.System.PottsSystem — Type
PottsSystem(; cell_types::Vector{CellType}, penalties::Vector{Any})

Collects the abstract logical rules and components defining a Cellular Potts Model.

PottsToolkit.System.SurfaceAreaComponent — Type
SurfaceAreaComponent(mappings::Pair{CellType, <:NamedTuple}...; eta=1.0)

Maps a CellType to its (target, λ) surface area parameters (hydrostatic formulation).

PottsToolkit.System.VolumeComponent — Type
VolumeComponent(mappings::Pair{CellType, <:NamedTuple}...)

Maps a CellType to its (target, λ) volume parameters for a standard harmonic spring penalty.

PottsToolkit.Problem.build_initial_state — Method
build_initial_state(u0_counts::Dict{CellType, Int}, 
                    type_to_id::Dict{CellType, UInt8}, 
                    initial_props::Dict{UInt8, Dict{Symbol, Any}},
                    grid_size::Tuple)

Constructs the grid and StructArray for the PottsState.

PottsToolkit.Problem.compile_component — Function
compile_component(pen::AbstractComponent, type_to_id::Dict{CellType, UInt8}, num_types::Int)

Returns (compiled_penalty, required_trackers, initial_props).
