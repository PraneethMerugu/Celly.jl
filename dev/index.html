Potts.jl

Potts.jl is a Cellular Potts modeling system built around two paper-core packages:

  • PottsToolkit is the biological authoring interface.
  • CorePotts is the scientific execution and extension interface.

The current development and evidence target is Julia 1.12.6.

Phase 13 evidence boundary

SequentialCPM implements the declared conventional sequential process. CheckerboardSweepCPM is a distinct graph-colored scheduler: it is characterized explicitly and does not inherit sequential kinetic, equilibrium, or stationary-distribution claims. Use CorePotts.algorithm_guarantees and CorePotts.scientific_contract_versions to inspect the machine-readable status attached to an algorithm and the versions embedded in evidence.

Backend execution support and scientific qualification are separate claims. A successful compatibility preflight does not imply that a backend is represented by the current Phase 13 evidence archive.

Scope

Lottery and tiled checkerboard algorithms are later protocol consumers and are not part of the initial Phase 13 paper-core qualification matrix. Visualization and neural-model satellites are deferred to Phase 14 or later and are not part of this API-freeze documentation.

Start with Getting Started, then consult the scientific contract identities and algorithm guarantees.
